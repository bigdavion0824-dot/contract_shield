## One-time setup in GitHub

1. Open repository settings.
2. Go to `Pages`.
3. Under `Build and deployment`, set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
4. Save.

GitHub usually publishes in 1-5 minutes.

If you still get a 404 after 10 minutes:

1. Confirm Pages source is `main` + `/docs`.
2. Make sure latest commit is on `main`.
3. In `Pages`, check deployment status and error details.
